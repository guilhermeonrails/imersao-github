Sobre o Curso

Neste curso completo, você aprenderá desde os fundamentos do Git até os recursos avançados do GitHub. Começaremos com a instalação e configuração do Git, passando pelos comandos básicos para versionar seus projetos, criar branches, gerenciar merges e resolver conflitos.

Aprenda a utilizar o GitHub para colaboração em equipe, gerenciamento de issues, pull requests, code reviews e muito mais. O curso é totalmente prático, com exemplos reais e exercícios para você fixar o aprendizado e aplicar no seu dia a dia como desenvolvedor.

  • Aulas 100% online e flexíveis
  • Material didático completo e atualizado
  • Suporte para dúvidas com o instrutor
  • Certificado de conclusão

Módulos do Curso

Módulo 1: Fundamentos do Git

Instalação, Configuração, Primeiros Comandos, Repositórios Locais, Staging Area, Commits, Histórico.

Módulo 2: Branching e Merging

Branches, Merge, Conflitos, Estratégias de Branching, Git Flow, GitHub Flow.

Módulo 3: Colaboração com GitHub

Repositórios Remotos, Pull Requests, Issues, Code Reviews, GitHub Actions, GitHub Pages.
